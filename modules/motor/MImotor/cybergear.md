# MI motor
小米电机cybergear（即代码中的MImotor）的ID出厂默认为0x7F（8位ID），要修改必须单独使用程序进行修改，修改代码在cybergear.c中也有

由于小米电机的运控模式自带PID计算，且电机是角度模式，速度模式，力矩模式是根据你是否给定Kp, Kd值而定
（1）角度模式：Kp != 0, Kd != 0
（2）速度模式：Kp == 0, Kd != 0
（3）力矩模式：Kp == 0, Kd == 0

我们的轮毂只需要使用力矩模式，因此初始化的时候不需要给定Kp, Kd，直接在程序运行过程给定目标力矩（12Nm~12Nm）即可
